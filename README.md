# EminnTech Portfolio 🌐

Personal portfolio and web design business site for Emmanuel Joshua — founder of EminnTech, based in Ibadan, Nigeria.

## What it is
A multi-page portfolio site built to showcase web development and automation work, convert visitors into clients, and establish EminnTech as a professional web services brand for Nigerian businesses.

## Pages
- **Home** — brand intro, services overview, CTA
- **Work** — 11 projects with filter tabs and zoom modal
- **About** — timeline, skills, values
- **Get Started** — Calendly booking + Netlify contact form

## Tech Stack
- HTML · CSS · JavaScript
- Netlify (hosting + forms)
- Calendly (booking integration)

## Design
- Deep navy + electric blue + cyan colour palette
- Mobile-first, fast-loading
- Pricing displayed in Naira
